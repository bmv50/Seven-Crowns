# Живые NPC (Utility AI + FSM)

Новый слой оживляет рядовых мобов: вне боя они бродят между комнатами, дремлют при усталости,
а раненый и напуганный моб может сбросить агро и убежать в соседнюю комнату. Идея адаптирована
из референс-движка `TeleMud/` (Go).

## Как включить
По умолчанию **выключено** (поведение игры не меняется). Включается переменной окружения —
так же, как `RULES_V2`:

```powershell
# временно в текущей сессии PowerShell:
$env:NPC_AI = "1"
python -m bot.main

# или добавь строку в файл .env:
NPC_AI=1
```

Выключить — убрать переменную или `NPC_AI=0`.

## Что меняется при включении
- Мобы вне боя иногда уходят в соседние локации (можно встретить «забрёл сюда»/«уходит прочь»).
- Уставшие мобы дремлют (восстанавливают «усталость»).
- Раненый моб (HP < 20%) при высоком страхе убегает: бросает бой и уходит.

Боёвка, лут, награды — не затронуты. Это надстройка «жизни мира».

## Тонкая настройка
Всё в `engine/npc_ai.py`:
- `RATE_HUNGER / RATE_FATIGUE / RATE_BOREDOM / DECAY_FEAR` — скорость роста/спада потребностей.
- `TICK_BUDGET` — сколько мобов максимум обрабатывать за тик (защита CPU).
- Вероятности блуждания/дрёмы — в `tick_ambient` (`random.random() < 0.12` и т.п.).
- Кривые полезности и пороги FSM — `evaluate_goal` / `step_fsm`.
