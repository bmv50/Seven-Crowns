# Мировые события и Сезоны

Два новых модуля, каждый за своим флагом (по умолчанию выключены).

## 🌐 Динамические мировые события (`WORLD_EVENTS=1`)
Раз в ~10 минут с шансом запускается случайное событие из `data/events.yaml`:
- **Нашествие из Пепла** — в Пепельных Пустошах спавнятся усиленные твари, опыт/золото ×1.5.
- **Ярмарка Семи Корон** — глобально опыт/золото ×1.25.
- **Разлом Глуби** — в Затонувшем Городе добыча ×2, опыт ×1.5.

Игроки получают рассылку о начале/конце; активное событие показывается баннером
в комнате и на экране «🌐 События» (меню Героя). По истечении вторженцы деспавнятся.

Включить:
```powershell
$env:WORLD_EVENTS = "1"
python -m bot.main
```
Свои события — в `data/events.yaml` (тип invasion/fair/anomaly, зона, множители,
длительность, моб для вторжения). Частоту/шанс — константы в `engine/events.py`
(`CHECK_EVERY`, `START_CHANCE`, `MAX_ACTIVE`).

## 🏅 Сезоны / лиги (`SEASONS=1`)
Сезон длится неделю (`SEASON_LENGTH` в `engine/seasons.py`). За убийства капают
сезонные очки (боссы ×5). Лиги по порогам: Бронза → Серебро → Золото → Платина →
Алмаз → Легенда. В конце сезона очки конвертируются в золотую награду по лиге,
лучший результат сохраняется, счёт обнуляется. Экран «🏅 Сезон» (меню Героя)
показывает лигу, очки, остаток времени и таблицу лидеров.

Включить:
```powershell
$env:SEASONS = "1"
python -m bot.main
```
Пороги лиг и награды — список `TIERS` в `engine/seasons.py`.
