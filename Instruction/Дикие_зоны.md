# Процедурные «дикие» зоны

Генератор дёшево набивает дикие окраины/подземелья вокруг рукотворного мира.
Каждая зона — связный граф комнат (2D random-walk + остовное дерево + петли),
детерминированный по seed. Адаптировано из референс-движка `TeleMud/`.

**Рукотворное ядро не трогается** — зоны лишь «подвешиваются» к существующим
комнатам и не перезаписывают контент.

## Предпросмотр (ничего не меняет в игре)
```powershell
python scripts/gen_wild_preview.py                     # дефолтная зона, ASCII-карта
python scripts/gen_wild_preview.py --region oldmine     # gloomwood | oldmine
python scripts/gen_wild_preview.py --size 24 --seed 7
```
Легенда карты: `@` вход · `M` комната с мобами · `o` пустая · `─│` проходы.

## Включить в игре
По умолчанию **выключено**. Включается переменной окружения (как RULES_V2 / NPC_AI):
```powershell
$env:WILD_ZONES = "1"     # или строка WILD_ZONES=1 в .env
$env:WILD_SEED = "42"     # необязательно: seed мира (один seed → один мир)
python -m bot.main
```
При старте к рукотворным «диким» комнатам подвесятся готовые зоны из
`engine/worldgen.py → WILD_ZONE_SPECS` (сейчас: Сумрачный Лес, Заброшенные Штольни).

## Свои зоны
В `engine/worldgen.py`, список `WILD_ZONE_SPECS`:
```python
RegionSpec(
    region_id="frostfen",          # уникальный id (войдёт в ключи комнат)
    name="Морозная Топь",          # отображаемое имя зоны
    biomes=["Наледь", "Полынья"],  # биомы для названий/описаний комнат
    size=30,                        # число комнат
    spawn_chance=0.55,              # шанс мобов в комнате
    mob_pool=[("волк", 3.0), ("порченый_лис", 1.0)],  # (mob_id, вес) — из mobs.yaml
)
```
