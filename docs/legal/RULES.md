# Правила сообщества «Семь Корон»

> ⚠️ **ЧЕРНОВИК — требует юридической проверки.** Правила описывают
> текущую (беташную) модерационную практику и техническую механику
> жалоб/банов. Формулировки мер и апелляции должны быть согласованы с
> юристом перед публикацией для реальных пользователей.

Дата последнего обновления черновика: [УКАЖИТЕ ДАТУ].

Эти правила действуют внутри Telegram-бота «Семь Корон» — в игровом
чате комнат, чате гильдии и общем групповом чате (если используется).

## 1. Имена персонажей

Имя персонажа должно быть уникальным и не должно:
- содержать оскорбления, нецензурную лексику, экстремистские или иначе
  незаконные призывы;
- имитировать имя администрации/модератора или вводить других игроков
  в заблуждение (фишинг, выдача себя за официальный аккаунт);
- содержать ссылки, рекламу сторонних сервисов.

Администрация вправе переименовать или сбросить персонажа с
нарушающим правила именем без предварительного уведомления.

## 2. Общение в чате

- Общение в чате комнаты/гильдии — только по теме игры и в уважительном
  тоне к другим игрокам.
- Запрещены: оскорбления, травля, разжигание конфликтов на почве
  расы/пола/религии/политики, спам и флуд, реклама сторонних проектов,
  попрошайничество реальных денег.
- Технический анти-спам лимит (ограничение частоты сообщений) действует
  автоматически и не заменяет модерацию по содержанию.

## 3. Честная игра

Запрещено:
- использование читов, автокликеров, эксплойтов багов игрового баланса
  для получения нечестного преимущества;
- **злоупотребление реферальной программой**: создание нескольких
  аккаунтов одним человеком для получения повторных наград за
  «приглашение друга» (мультиаккаунтинг), обмен взаимными фиктивными
  приглашениями, накрутка счётчика приглашений любым иным способом;
- перепродажа/передача игрового аккаунта третьим лицам;
- манипуляции с аукционом/торговлей в сговоре для увода игровой валюты в
  обход честной экономики.

## 4. Как пожаловаться

Если другой игрок нарушает правила — используйте кнопку
**«⚠️ Пожаловаться»** рядом с именем игрока в списке игроков комнаты
(экран группы/окружения). Жалоба фиксируется в журнале и уходит
администраторам на разбор. Также можно написать напрямую в поддержку
командой `/support`.

Пожалуйста, указывайте по возможности, что именно произошло — это
ускоряет разбор.

## 5. Лестница мер

Меры применяются по усмотрению администрации с учётом тяжести и
повторности нарушения, обычно по нарастающей:

1. **Предупреждение** — устное/текстовое замечание без ограничения
   доступа.
2. **Мут** — временный запрет писать в чат (минуты/часы, зависит от
   тяжести нарушения).
3. **Бан** — полная блокировка доступа к игре (временная или
   постоянная — по решению администрации).

Для явных нарушений (читы, фишинг, незаконный контент) администрация
вправе сразу перейти к более строгой мере в обход предыдущих ступеней.

[УКАЖИТЕ: точные сроки мутов/банов по категориям нарушений, если
требуется зафиксировать формально].

## 6. Апелляция

Если вы считаете меру ошибочной — обратитесь через команду `/support` с
описанием ситуации. Администрация рассмотрит обращение в срок
[УКАЖИТЕ СРОК РАССМОТРЕНИЯ].
