# Семь Корон — предложения по игровым механикам

Аналитика на основе популярных MMORPG (WoW, Lineage 2, RuneScape/OSRS, Path of Exile,
Albion Online, Diablo, ESO/GW2, Tibia) с поправкой на формат Telegram-MUD
(текст + кнопки, асинхронный мир, короткие сессии). Для каждой механики: суть,
аналог-источник, как ложится на наш движок, оценка усилий, приоритет.

Легенда приоритета: 🟢 взять скоро · 🟡 средне · 🔵 потом/опционально.
Уже в плане проекта: телепорт, питомцы, добыча/крафт/алхимия, реморт, гильдии,
PvP-флаг/арена/стражи, банк/гостиница, Stars. Ниже — что добавить сверх этого и как.

---

## 1. Прогрессия и билды

**🟢 Дерево талантов / специализации.** На каждом N уровней — очко таланта; ветки
усиливают стиль (танк/урон/поддержка). Источник: WoW talent trees, PoE passive tree
(упрощённо). У нас уже есть классы и ресурсы — таланты дают «билды» без новых классов.
Усилия: средние (данные talents.yaml + экран распределения). Сильно повышает реиграбельность
и завязывается на реморт (сохранять 1–2 таланта).

**🟢 Редкость предметов и аффиксы.** Обычный/необычный/редкий/эпический/легендарный, с
случайными бонусами (например «+3 Силы», «+5% крит»). Источник: Diablo, PoE. У нас лут
фиксированный — добавить «тиры лута» (то, что ты просил для красных мобов): красные/боссы
роняют предметы выше редкостью со случайными аффиксами. Усилия: средние. Закрывает запрос
«лут более высокого уровня».

**🟡 Зачар/сокеты/руны.** Улучшение снаряжения камнями/рунами с риском (L2: при провале
вещь слабеет/ломается — острые эмоции и золотосток). Источник: Lineage 2, PoE sockets.
Хороший денежный сток. Усилия: средние.

**🟡 Прочность и ремонт.** Снаряжение изнашивается, ремонт у кузнеца за деньги. Источник:
WoW durability. Мягкий штраф за смерть и сток золота. Усилия: малые.

## 2. Контент-петли (что делать каждый день)

**🟢 Подземелья-инстансы (данжи).** Связка из 3–6 комнат с мини-боссом в конце, частный
инстанс на группу, кулдаун входа. Источник: WoW dungeons. У нас уже есть комнаты, бой,
пати, трупы — данж это «помеченная цепочка комнат + награда». Усилия: средние. Главный
источник кооп-контента.

**🟢 Ежедневные/еженедельные задания.** Ротация коротких квестов («убей 10 X», «сдай 5 руды»)
с бонус-наградой. Источник: WoW dailies, мобильные MMO. Удержание + причина заходить.
Усилия: малые (поверх системы квестов + scheduled-tasks для ротации).

**🟡 Мировые боссы по таймеру.** Раз в N часов в зоне появляется супер-моб; кто участвовал —
получает долю награды. Источник: GW2/Lineage 2 raid bosses. Социальное событие. Усилия: средние.

**🔵 Динамические события «ИИ-бога».** Случайные события в зонах (нашествие, ярмарка, аномалия).
Источник: GW2 dynamic events. Уже в твоих заметках. Усилия: средние-высокие.

**🟡 Бестиарий/коллекции.** За убийство видов мобов — записи в бестиарий + пассивные бонусы
за полноту. Источник: Witcher-стиль, achievement-системы. Дешёвое удержание. Усилия: малые.

## 3. Экономика

**🟢 Аукцион / торговая площадка.** Игроки выставляют предметы, другие покупают за золото.
Источник: WoW Auction House, RuneScape Grand Exchange. Оживляет крафт и добычу, создаёт
рынок. Усилия: средние (асинхронно, через БД). Очень ценно для долгоиграющей экономики.

**🟢 Золото-стоки.** Ремонт, обучение умений, телепорт, зачар, аукционный сбор, аренда комнаты
в гостинице. Без стоков экономика «инфлирует». Источник: любая MMO. Часть уже есть (телепорт/
обучение). Усилия: малые, по мере добавления систем.

**🟡 Узлы добычи и профессии-навыки.** Отдельная прокачка добычи (рудокоп/травник/рыбак/
охотник) с уровнями навыка, как классы у RuneScape. Источник: OSRS/RuneScape, Albion. У тебя
уже запланированы гильдии добычи — связать с уровнями навыка и рецептами. Усилия: средние.

**🔵 Локальные рынки (риск перевозки).** Цены отличаются по городам; возить товар выгодно, но
опасно (засады). Источник: Albion Online. Глубоко, но нишево. Усилия: высокие.

## 4. Социальное и PvP

**🟢 Гильдии/кланы с банком и рангами.** Общий склад, ранги, гильд-чат, гильд-задания.
Источник: все MMO. У нас есть пати — клан это «постоянная пати + хранилище + прогресс».
Усилия: средние. Сильное удержание.

**🟢 Рейтинговая арена (ладдер).** Очки рейтинга за победы 1v1/2v2, сезоны, награды по рангу.
Источник: WoW Arena. У нас уже есть дуэли и арена-комната — добавить рейтинг/очередь. Усилия:
средние.

**🟡 Карма/refine PvP-флага.** Развить твою «красную метку»: убийство мирного даёт карму;
высокая карма — стражи агрятся, при смерти роняешь вещи, NPC не торгуют. Источник: Lineage 2
karma/PK, Tibia skull system. Делает открытый PvP «со ставками». Усилия: средние.

**🟡 Зоны открытого PvP / фракционная война.** Вне городов — опасные зоны с PvP и лучшим лутом;
война фракций за контроль территорий. Источник: Albion full-loot zones, ESO Cyrodiil. Усилия:
средние-высокие.

## 5. Удержание и «мета»

**🟢 Достижения и титулы.** «Убей первого босса», «100 крафтов», «достигни 50 ур.» → титул
к имени + мелкая награда. Источник: WoW achievements. Дешёво, цепляет. Усилия: малые.

**🟢 Репутация с фракциями.** Действия повышают/понижают репутацию; высокая открывает
магазины/квесты/скидки. Источник: WoW reputations. У нас уже есть фракции у NPC — добавить шкалу.
Усилия: средние.

**🟡 Маунты и питомцы с прогрессией.** Питомцы (уже в плане) растут, маунты ускоряют
телепорт/перемещение и косметика за Stars. Источник: WoW mounts/pets. Усилия: средние.

**🔵 Жильё/гильд-холл.** Личная комната (уже есть в гостинице) → улучшаемое жильё с бонусами
отдыха/хранения. Источник: ESO/FFXIV housing. Усилия: высокие.

**🔵 Сезоны/лиги.** Временные серверные сезоны со своими правилами и таблицей лидеров; по
окончании — награды. Источник: PoE leagues, Diablo seasons. Усилия: высокие, но мощное удержание.

## 6. Углубление боёвки

**🟡 Угроза/аггро для танков.** Танк-умения генерируют угрозу, мобы бьют по верхней угрозе —
осмысленная «троица» танк/дамаг/хил. Источник: WoW threat. У нас уже есть классы под роли —
это «оживит» кооп. Усилия: средние.

**🟡 Статусы и комбо.** Поджог/заморозка/оглушение/кровотечение со взаимодействиями
(«заморозил → следующий удар крит»). Источник: GW2 combo fields, PoE ailments. Усилия: средние.

**🟢 Бафф-еда и зелья-усиления.** Расходники из крафта/алхимии дают временные бонусы перед
боем. Источник: WoW food/flasks. Связывает алхимию с боёвкой. Усилия: малые.

## 7. Качество жизни (QoL)

**🟢 «Отдохнувший» опыт.** Пока офлайн/в гостинице копится бонус к опыту (×2 на N убийств).
Источник: WoW rested XP. Поощряет заходить и пользоваться гостиницей. Усилия: малые.

**🟢 Трекер квестов и автопуть.** Активная цель показывается в комнате; «идти к цели»
прокладывает маршрут по кнопкам/командам. Источник: современные MMO. Усилия: малые-средние.

**🟢 Пресеты лоудаута/экипировки.** Сохранённые наборы умений+снаряжения под роль (бой/добыча).
Источник: WoW equipment manager. Усилия: малые.

## Рекомендованный шорт-лист (баланс «ценность/усилия»)

Если брать 6 механик, которые сильнее всего улучшат игру при разумных усилиях:

1. **Редкость + аффиксы лута** (закрывает «лут получше с красных», даёт охоту за вещами).
2. **Подземелья-инстансы** (главный кооп-контент поверх уже готовых систем).
3. **Аукцион/рынок** (оживляет крафт, добычу и экономику).
4. **Дерево талантов** (билды и глубина без новых классов; связь с ремортом).
5. **Достижения + ежедневки + отдохнувший опыт** (дешёвое удержание, причина заходить).
6. **Угроза/аггро + бафф-еда** (осмысленная «троица» и связь алхимии с боем).

Это аккуратно ложится поверх запланированных фаз (телепорт/питомцы/добыча/крафт/реморт/
гильдии/PvP) и не требует ломать движок: почти всё — это новые YAML-данные + экраны-кнопки.
